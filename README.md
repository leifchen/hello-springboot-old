# Hello-SpringBoot

整理常用 Spring Boot 相关的知识内容。

## 多环境配置

- 配置开发环境：application-dev.yml
- 配置正式环境：application-prd.yml

借由配置文件 application.yml 中的 spring.profiles.active=dev 来实现环境的选择。
